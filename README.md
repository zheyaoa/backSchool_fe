# 项目结构

> assets 静态文件(不更换)
>
> static  静态文件(需要更换)
>
> compoents 一些公共组件
>
> /unit 一些公共资源
>
> > /api   api 函数
>
> views  页面视图(每一个页面一个文件夹)
>
> > /home  根目录
> >
> > > /index.vue 导出模块
> > >
> > > /others  home页面下的其他模块
>
> route.js 路由文件
>
> store .js 代码仓库
>
> App.vue 项目根页面
>
> main.ts 项目根入口
>
> *.d.ts   ts 声明文件
